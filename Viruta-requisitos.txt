Lista de Requisitos:
RF1 - El sistema debería guardar un registro de los viajeros y de los revisores.
RF2 - El sistema debería tener acceso a los horarios y recorridos de cada tren
RF3 - El sistema debería tener acceso a los registros de tarifas de descuentos de calendario
RF4 - El sistema debería admitir varios métodos de pago
RF5 - El sistema debería comprobar si hay alguna clase de descuento para el viaje seleccionado.
RF6 - El sistema debería hacer la venta de billetes para distintos trayectos y horarios por separado
RF7 - El sistema debería elaborar un justificante de la factura del viaje
RF8 - El sistema debería permitir descargar los datos de las ventas realizadas cuando el revisor se identifique.
